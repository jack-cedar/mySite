Html Canvas 3D Renderer

This is a 3d Renderer I wrote with Html Canvas 2d Context (Not WebGL)
after alot of resarch into the math behind it and is uses no libraries
It is capable of Perspective projection, scaling, translation, rotation etc.
I mostly made it to learn about matrix math and how a 3d object can be rendered on a 2d surface.
Never really ended up finishing it
It also includes a very rudimentary parser for .obj files in the utils.js file
